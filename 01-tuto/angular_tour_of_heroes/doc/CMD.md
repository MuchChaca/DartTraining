# Commands

## Install
[github.com/google/angular_cli](https://github.com/google/angular_cli#installation)

``ngdart -h``

## Create a new app
```bash
# Create new app
> ngdart new [project_name]
# Go to the project directory
> cd [project_name]
# Get dependencies
> pub get
```

## Generate tests
```bash
> ngdart generate test lib/app_component.dart
```
